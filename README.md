# E-commerce Profit Intelligence Dashboard

An end-to-end Microsoft Excel analytics project focused on identifying profit leakage, evaluating customer behavior, and supporting executive decision-making through interactive dashboards.

---

## Project Overview

This project analyzes an e-commerce business experiencing strong sales growth but limited profitability. Using Microsoft Excel, it transforms raw transactional data into interactive dashboards that provide actionable business insights. The project covers the complete analytics workflow, from data preparation to dashboard creation, with a focus on supporting data-driven decision-making.

---

## Project Scope

This repository documents the complete development process of the dashboard, including:

- Data collection

- Data cleaning

- Feature engineering

- KPI development

- Dashboard design

- Business analysis

- Strategic recommendations
